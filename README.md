# hgnc

## Purpose
The purpose of this project is to download data from the HGNC website and convert it to a readable database format.
This allows users to query the data and returns the filtered dataset to support the process of analyzing genes.

## Features
- Download data from URL and convert to readable database
- Query for values in each data category and return filtered dataset
- Support for data with multiple values

## Data

The following data is stored in the database:

1. ID
2. Approved symbol
3. Approved name
4. Status
5. Previous symbols
6. Alias symbols
7. Chromosome
8. Accession numbers
9. RefSeq IDs
